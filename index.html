Rusty wayland

This page regroups ressources about project of the Smithay ecosystem, providing tools for using wayland in Rust.

The Rusty Wayland Handbook. A currently WIP general tutorial about wayland and how to use the various libraries.

wayland-client API docs. API documentation of the wayland-client crate. It is the core crate for writing wayland client apps.

wayland-kbd API docs. API documentation of the wayland-kbd crate. It is an helper crate for wayland clients to handle keymaps for keyboard inputs.

wayland-window API docs. API documentation of the wayland-window crate. It is an helper crate for wayland clients to easily draw decorations and handle resizing/move.

wayland-server API docs. API documentation of the wayland-server crate. It is the core crate for writing wayland compositors.

smithay API docs. API documentation of the smithay crate, a WIP toolkit for writing wayland compositors.
